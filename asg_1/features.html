Assignment 1: Basic Javascript/WebGL, Polylines


Assignment 1

Live Hosted

    Required

  1. Have an HTML page with a canvas element.
  2. Detect mouse clicks (left and right) and echo them to debugging console.
  3. Clicked points are displayed on the canvas.
  4. Lines are drawn between displayed points in the order that they were clicked.
  5. A line is drawn between the last clicked point and the current cursor position (the rubber band line).
  6. When the user right clicks, hide the rubber band line, block further user input, and print a list of the clicked points (in the order they were clicked) to the console.

    7. Extras

  7. The user can change the colors of the polylines by clicking in certain quadrents of the canvas.
  8. The user can change the size of the points by adjusting the slider.
