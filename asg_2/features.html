Assignment 2: Generalized Cylinder Wireframe


Assignment 2

Live Hosted

    Required

  1. (REQUIRED) Assignment draws a fixed generalized cylinder. By default, the generalized cylinder should have 12 sides.
  2. (REQUIRED) If the user just draws a single straight line, a generalized cylinder is drawn around it.
  3. (REQUIRED) If the user draws a polyline, generalized cylinders are drawn around each line segment.
  4. (REQUIRED) The user can save the model currently on screen to an OBJ file.
  5. (REQUIRED) The user can load and display a model from an OBJ file.

    6. Extras

  6. (EXTRA) Intersections between sections of your generalized cylinder are covered so that there are no "gaps."
  7. (EXTRA) The user can have multiple cylinders on the screen at once.
  8. (EXTRA) The user can change the number of sides the cylinder has with a slider.
  9. (EXTRA) The user can change the width of the cylinder with a slider (i.e. the radius of the polygonal face).
